Azure App Registration
Register an app in Microsoft Entra ID for Graph API access and (optionally) SSO.
Setup Instructions
  1. Go to Entra ID → App registrations and click New registration.
  2. Name: CCompliance, Single tenant, Redirect URI type Web:
    Also add a second Redirect URI:
  3. Click Register. Copy the Application (client) ID and Directory (tenant) ID.
  4. Go to Certificates & secrets → New client secret. Copy the Value (not the Secret ID).
  5. Add API permissions → Microsoft Graph → Application permissions: User.Read.All Mail.ReadWrite Mail.Send
Anthropic Compliance API
Connect to your organization's Anthropic Compliance API for activity and chat data.
Where to find your key
  1. Log in to console.anthropic.com
  2. Navigate to Settings → Compliance API.
  3. Generate or copy your Compliance Access Key.
Azure Key Vault Optional
Store API keys securely in Azure Key Vault using managed identity. You can skip this step.
Setup Instructions
  1. In the Azure Portal, create a new Key Vault (or use an existing one).
  2. In your Web App → Identity, ensure System assigned managed identity is On.
  3. In your Key Vault → Access control (IAM), add a role assignment: Key Vault Secrets User → assign to your Web App's managed identity.
  4. Paste the Key Vault URL below and click Test Connection.
